先记住英语顺序

英语不是把单词一个一个乱放。最基础的顺序是：谁 + 做什么/是什么 + 什么东西/地点/时间。

例子：I need water. = I（我）+ need（需要）+ water（水）。学词库时，不只看单词意思，还要看它在句子里的位置。

1. 先懂顺序英语先说“谁”，再说“做什么/是什么”。
2. 先会求助I need help. / Please speak slowly.
3. 再学生活点餐、购物、问路、看病、住宿。
4. 最后复习用选择题和听力题检查是否记住。

图片词汇

只给适合看图理解的生活单词加图片：食物、交通、地点、日用品、身体、标识等。抽象词和长句不强行配图，避免误导。

学习方法：先看图猜英文，再点“听读”；最后去“复习测试”做“看图选英文、英文选图、听音选图”。
使用方法：先点“听读”，跟着慢慢读；能理解并会说，就点“标记已学”；生活里经常会用的句子点“收藏”。

语速

发音打不开怎么办？

  1. 先点页面右上角“测试发音”。很多浏览器必须用户先点一次按钮，才允许播放声音。
  2. 手机或电脑不要静音，音量调大。
  3. 建议使用 Chrome、Edge 或 Safari。
  4. 如果某个声音很奇怪，在上方换一个英文声音。
  5. 手机浏览器第一次加载英文声音可能要等几秒钟。
发音练习方法：先点“更慢”，听一遍；再自己跟读一遍；最后点“正常”，听真实速度。
