先记住英语顺序

英语不是把单词一个一个乱放。最基础的顺序是：谁 + 做什么/是什么 + 什么东西/地点/时间。

例子：I need water. = I（我）+ need（需要）+ water（水）。学词库时，不只看单词意思，还要看它在句子里的位置。

第一步：会求助

I need help. / Please speak slowly. / Can you write it down?

第二步：会生活

点餐、购物、问路、看病、住宿、手机表格。

第三步：会复习

点“已学”后会自动加入复习；答错会更快再出现。

  3. 5分钟：复习昨天点过“已学”的内容。
  4. 2分钟：大声说今天最有用的3句话。
使用方法：先点“听读”，跟着慢慢读；能理解并会说，就点“标记已学”；生活里经常会用的句子点“收藏”。
